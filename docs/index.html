msdss-base-dotenv

Environmental file management for the Modular Spatial Decision Support Systems (MSDSS) framework.

  • PyPi

  • Github

  • License

Install

  1. Install Anaconda 3 for Python

  2. Install msdss-base-dotenv via pip or through a conda environment

conda create -n msdss-base-dotenv python=3.8
conda activate msdss-base-dotenv
pip install msdss-base-dotenv

Quick Start

After installing the package, use in Python via functions:

 1from msdss_base_dotenv import save_env_file, load_env_file, clear_env_file, env_exists
 2
 3# Clear existing env files
 4clear_env_file()
 5
 6# Check if environment exists
 7exists_before = env_exists()
 8
 9# Create key-value environmental variables to save
10env = dict(user='msdss', password='msdss123')
11
12# Save the environmental variables as an encrypted file
13save_env_file(env, file_path='./.env')
14
15# Load the encrypted environmental variables
16loaded_env = load_env_file(file_path='./.env')
17
18# Load with defaults
19defaults = dict(database='postgres', port='5432')
20loaded_env_defaults = load_env_file('./.env', defaults=defaults)
21
22# Check if environment exists after creation
23exists_after = env_exists()
24
25# Display the results
26print('exists_before: ' + str(exists_before))
27print('env: ' + str(env))
28print('loaded_env: ' + str(loaded_env))
29print('loaded_env_defaults: ' + str(loaded_env_defaults))
30print('exists_after: ' + str(exists_after))

exists_before: False
env: {'user': 'msdss', 'password': 'msdss123'}
loaded_env: {'user': 'msdss', 'password': 'msdss123'}
loaded_env_defaults: {'database': 'postgres', 'port': '5432', 'user': 'msdss', 'password': 'msdss123'}
exists_after: True

msdss_base_dotenv.core.clear_env_file(file_path='./.env', key_path=None)[source]

Deletes the encrypted environment file and its key if they exist.

Parameters

  • file_path (str) – Path of the encrypted environment file.

  • key_path (str) – Path of the key file used to unlock the encrypted file. If None, this defaults to the package’s directory.

Author

Richard Wen <rrwen.dev@gmail.com>

msdss_base_dotenv.core.load_env_file(file_path='./.env', key_path=None, defaults={})[source]

Loads a saved environment file from msdss_base_dotenv.core.save_env_file().

Parameters

  • file_path (str) – Path of the environment save file.

  • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

  • defaults (dict) – Key and value pairs representing default environment values to be loaded. These will replace ones in env if they do not exist or are unset.

Returns

Dictionary of the decrypted key value environment from file_path.

Return type

dict

Author

msdss_base_dotenv.core.save_env_file(env, file_path='./.env', key_path=None, defaults={})[source]

Saves a login file with the connection details.

Parameters

  • env (dict) – Key and value pairs representing environment values to be saved.

  • file_path (str) – Path of the encrypted environment save file. Read/write/execute permissions will be set only for the owner of this file.

  • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

  • defaults (dict) – Key and value pairs representing default environment values to be saved. These will replace ones in env if they do not exist or are unset.
