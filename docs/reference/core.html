    _dict_to_dotenv¶

    msdss_base_dotenv.core._dict_to_dotenv(env_dict)[source]¶

    Converts a dict to a .env formatted str.

    Parameters

    env_dict (dict) – Key value dictionary representing env variables.

    Returns

    Str representing a .env formatted file structure.

    Return type

    str

    Author

    Richard Wen <rrwen.dev@gmail.com>

    Example

     1from msdss_base_dotenv.core import _dict_to_dotenv
 2
 3# Create default key value env
 4env = dict(USER='msdss', PASSWORD='msdss123')
 5
 6# Convert to dotenv file str
 7env_str = _dict_to_dotenv(env)
 8
 9# Display results
10print('env: ' + str(env))
11print('env_str: ' + env_str)

    env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}
env_str: USER=msdss
PASSWORD=msdss123

10clear_env_file()

    del_env_var¶

    msdss_base_dotenv.core.del_env_var(name, file_path='./.env', key_path=None)[source]¶

    Deletes an environmental variable using a file from msdss_base_dotenv.core.save_env_file().

    Parameters

    • name (str) – The name of the environmental variable to be deleted.

    • file_path (str) – Path of the environment save file.

    • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

    Author

    Richard Wen <rrwen.dev@gmail.com>

    Example

     1import os
 2
 3from msdss_base_dotenv.core import save_env_file, load_env_file, clear_env_file, del_env_var
 4
 5# Clear any existing env files
 6clear_env_file()
 7
 8# Create key value env and save it
 9env = dict(USER='msdss', PASSWORD='msdss123')
10save_env_file(env)
11
12# Load the saved env
13load_env_file()
14
15# Remove the password var from the saved env
16del_env_var('PASSWORD')
17
18# Load the saved env after the removal
19load_env_file()
20loaded_env = dict(
21    USER=os.environ['USER'],
22    PASSWORD=os.getenv('PASSWORD', None)
23)
24
25# Display the results
26print('env: ' + str(env))
27print('loaded_env: ' + str(loaded_env))

    env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}
loaded_env: {'USER': 'msdss', 'PASSWORD': None}

    env_exists¶

    • file_path (str) – Path of the environment save file.

    • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

    Whether or not file_path and key_path exist, which define whether the env exists.

    Return type

    bool

    Author

    Richard Wen <rrwen.dev@gmail.com>

 5
 6# Check if env exists

 8
 9# Create key value env and save it

12

15
16# Display the results

exists_after: True

    load_env_file¶

    msdss_base_dotenv.core.load_env_file(file_path='./.env', key_path=None, defaults={}, set_env=True, return_dict=False)[source]¶

    Loads a saved environment file from msdss_base_dotenv.core.save_env_file().

    Parameters

    • file_path (str) – Path of the environment save file.

    • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

    • defaults (dict) – Key and value pairs representing default environment values to be loaded. These will replace ones in env if they do not exist or are unset.

    • set_env (bool) – Whether to set the os.environ with the variables in the file_path or not.

    • return_dict (bool) – Whether to return a dictionary of the env variables or not.

    Returns

    Dictionary of the decrypted key value environment from file_path if return_dict is True.

    Return type

    dict

    Author

    Richard Wen <rrwen.dev@gmail.com>

    Example

     1import os
 2
 3from msdss_base_dotenv.core import save_env_file, load_env_file, clear_env_file
 4
 5# Clear any existing env files
 6clear_env_file()
 7
 8# Create key value env and save it with defaults
 9env = dict(USER='msdss', PASSWORD='msdss123')
10defaults = dict(DATABASE='postgres', PASSWORD='already-set')
11save_env_file(env, defaults=defaults)
12
13# Load the saved env file
14load_env_file()
15loaded_env = dict(
16    USER=os.environ['USER'],
17    PASSWORD=os.environ['PASSWORD'],
18    DATABASE=os.environ['DATABASE']
19)
20
21# Display the results
22print('env: ' + str(env))
23print('defaults: ' + str(defaults))
24print('loaded_env: ' + str(loaded_env))

    env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}
defaults: {'DATABASE': 'postgres', 'PASSWORD': 'already-set'}
loaded_env: {'USER': 'msdss', 'PASSWORD': 'msdss123', 'DATABASE': 'postgres'}

    save_env_file¶

    msdss_base_dotenv.core.save_env_file(env, file_path='./.env', key_path=None, defaults={})[source]¶

    Saves a login file with the connection details.

    Parameters

    • env (dict) – Key and value pairs representing environment values to be saved.

    • file_path (str) – Path of the encrypted environment save file. Read/write/execute permissions will be set only for the owner of this file.

    • key_path (str) – Path of the key file used to unlock the save file. If None, this defaults to the package’s directory.

    • defaults (dict) – Key and value pairs representing default environment values to be saved. These will replace ones in env if they do not exist or are unset.

    Author

    Richard Wen <rrwen.dev@gmail.com>

    Example

     1from msdss_base_dotenv.core import save_env_file, load_env_file, clear_env_file
 2
 3# Clear any existing env files
 4clear_env_file()
 5
 6# Create default key value env
 7env = dict(USER='msdss', PASSWORD='msdss123')
 8
 9# Save the key value env to an encrypted file
10save_env_file(env)
