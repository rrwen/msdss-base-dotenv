msdss-base-dotenv

Environmental file management for the Modular Spatial Decision Support Systems (MSDSS) framework.

  • PyPi

  • Github

  • License

Install

  1. Install Anaconda 3 for Python

  2. Install msdss-base-dotenv via pip or through a conda environment

conda create -n msdss-base-dotenv python=3.8
conda activate msdss-base-dotenv
pip install msdss-base-dotenv

Quick Start

Initialize your env file and key in a command line terminal:

>>> msdss-dotenv init

Set env variables:

>>> msdss-dotenv set USER msdss
>>> msdss-dotenv set PASSWORD msdss123

Remove env variables:

>>> msdss-dotenv set TYPO randomtypo
>>> msdss-dotenv del TYPO

Clear env file and key:

>>> msdss-dotenv clear

Get help:

>>> msdss-dotenv --help
>>> msdss-dotenv set --help
>>> msdss-dotenv del --help

Usage

This package can also be used programmatically in Python via functions.

Saving and Loading Environment Variables

Create and load encrypted environment variables:

 1import os
 2
 3from msdss_base_dotenv import *
 4
 5# Clear existing env files
 6clear_env_file()
 7
 8# Check if env exists
 9exists_before = env_exists()
10
11# Save encrypted env vars
12env = dict(USER='msdss', PASSWORD='msdss123')
13save_env_file(env, file_path='./.env')
14
15# Load encrypted env vars
16load_env_file(file_path='./.env')
17loaded_env = dict(
18   USER=os.environ['USER'],
19   PASSWORD=os.environ['PASSWORD']
20)
21
22# Load env vars with defaults
23defaults = dict(DATABASE='postgres', PASSWORD='already-set')
24load_env_file('./.env', defaults=defaults)
25loaded_env_defaults = dict(
26   USER=os.environ['USER'],
27   PASSWORD=os.environ['PASSWORD'],
28   DATABASE=os.environ['DATABASE']
29)
30
31# Check if env exists after saving vars
32exists_after = env_exists()
33
34# Display the results
35print('exists_before: ' + str(exists_before))
36print('\n\nenv: ' + str(env))
37print('\n\nloaded_env: ' + str(loaded_env))
38print('\nloaded_env_defaults: ' + str(loaded_env_defaults))
39print('\n\nexists_after: ' + str(exists_after))

exists_before: False


env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}


loaded_env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}

loaded_env_defaults: {'USER': 'msdss', 'PASSWORD': 'msdss123', 'DATABASE': 'postgres'}


exists_after: True

Editing Environment Variables

Edit saved encrypted environment variable files:

 1import os
 2
 3from msdss_base_dotenv import *
 4
 5# Clear existing env files
 6clear_env_file()
 7
 8# Save env vars
 9env = dict(USER='msdss', PASSWORD='msdss123')
10save_env_file(env, file_path='./.env')
11
12# Load env vars
13load_env_file(file_path='./.env')
14
15# Remove the password variable
16del_env_var('PASSWORD')
17
18# Set a secret variable
19set_env_var('SECRET', 'some-secret')
20
21# Load the env vars with edits
22load_env_file(file_path='./.env')
23edited_env = dict(
24   USER=os.environ['USER'],
25   SECRET=os.environ['SECRET']
26)
27
28# Display the results
29print('env: ' + str(env))
30print('edited_env: ' + str(edited_env))

env: {'USER': 'msdss', 'PASSWORD': 'msdss123'}
edited_env: {'USER': 'msdss', 'SECRET': 'some-secret'}
